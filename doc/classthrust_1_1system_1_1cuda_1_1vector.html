Detailed Description

template<typename T, typename Allocator = allocator<T>>
class thrust::system::cuda::vector< T, Allocator >

cuda::vector is a container that supports random access to elements, constant time removal of elements at the end, and linear time insertion and removal of elements at the beginning or in the middle. The number of elements in a cuda::vector may vary dynamically; memory management is automatic. The elements contained in a cuda::vector reside in memory available to the cuda system.
