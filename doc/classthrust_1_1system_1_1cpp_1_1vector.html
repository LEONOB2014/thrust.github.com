Detailed Description

template<typename T, typename Allocator = allocator<T>>
class thrust::system::cpp::vector< T, Allocator >

cpp::vector is a container that supports random access to elements, constant time removal of elements at the end, and linear time insertion and removal of elements at the beginning or in the middle. The number of elements in a cpp::vector may vary dynamically; memory management is automatic. The elements contained in a cpp::vector reside in memory available to the cpp system.
