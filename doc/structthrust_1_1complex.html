Detailed Description

template<typename T>
struct thrust::complex< T >

complex is the Thrust equivalent to std::complex. It is functionally equivalent to it, but can also be used in device code which std::complex currently cannot.

Template Parameters
TThe type used to hold the real and imaginary parts. Should be float or double. Others types are not supported.
