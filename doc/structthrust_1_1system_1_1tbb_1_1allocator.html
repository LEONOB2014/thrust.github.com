Detailed Description

template<typename T>
struct thrust::system::tbb::allocator< T >

tbb::allocator is the default allocator used by the tbb system's containers such as tbb::vector if no user-specified allocator is provided. tbb::allocator allocates (deallocates) storage with tbb::malloc (tbb::free).
